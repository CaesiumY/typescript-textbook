# 🔨기본 셋팅

## 각 챕터 시작 전

```bash
npm init

npm install -D typescript eslint @types/node

tsc --init
```

## 폴더 구성

```
├────node_modules/
├────src/
│ └────index.ts
├────package.json
├────tsconfig.json
└────tslint.json
```

## 실행 명령어

```bash
npx tsc
```
